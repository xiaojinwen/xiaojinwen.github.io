## 问题
有时候我们获取的数据是一个数组对象,但是提交数据的时候又要你把数组转为对象其中还要筛选没用的key

### 例:
```bash
[
	{ name1: '我的', phone1: '122222', ttt1: '迟到了吗' },
	{ name2: '我的', phone2: '122222', ttt2: '迟到了吗' },
	{ name3: '我的', phone3: '122222', ttt3: '迟到了吗' }
]
处理为

const obj ={ 
	name1: '我的', phone1: '122222',
	name2: '我的', phone2: '122222', 
	name3: '我的', phone3: '122222' 
}
  
```

```bash
// filter 为筛选 项(即数组中的符合项)
// map 为 处理每一项中 所需的内容(key)
const arr = [
	{ name1: '我的', phone1: '122222', ttt1: '迟到了吗' },
	{ name2: '我的', phone2: '122222', ttt2: '迟到了吗' },
	{ name3: '我的', phone3: '122222', ttt3: '迟到了吗' }
]
let arr1 = arr.map((item, index) => {
	let obj = {}
	obj[`name${index + 1}`] = item[`name${index + 1}`]
	obj[`phone${index + 1}`] = item[`phone${index + 1}`]
	return obj
})
// console.log(arr1)
let obj = Object.assign(...arr1)
console.log(obj)
```
