### 一、使用`float:left; 或 float:right`

这个方法水平排列起来简单 只需
```bash
float:left;
```
但是会引起一些不必要的问题: 父级盒子没有float部分的高度,因为float脱离了标准文档流
解决办法可以设置父级的高,清理浮动,清理浮动的方式有很多,推荐使用伪类清理法
```bash
<div class ='clearfix'></div>

.clearfix::after {
	content: ".";
	clear: both;
	display: block;
	overflow: hidden;
	font-size: 0;
	height: 0;
}
// 为兼容IE6,IE7，因为ie6,ie7不能用after伪类。加上下面代码
.clearfix{zoom:1}
```
### 二、设置元素为行内块元素

在需要水平排列的地方
```bash
display: inline-block;
```

### 三、使用flex布局

在需要水平排列的父布局中

```bash
display: flex;
// flex-wrap: wrap; /*超出宽度强制换行*/
```
在子布局中

```bash
flex: 1;
// 更多参数选择请自行百度 flex
```
